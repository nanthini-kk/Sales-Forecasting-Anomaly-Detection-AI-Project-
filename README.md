##Power BI Business Analytics Dashboard

This project is an interactive Power BI dashboard designed to analyze business performance using key metrics, trends, and AI-driven insights. It helps stakeholders make data-driven decisions by visualizing revenue, profit, customer behavior, regional performance, and anomalies.

##Features

 - KPI cards for Total Revenue, Total Profit, Orders, and Customers

 - Profit and sales analysis across regions and categories

 - Time-based analysis using YTD, MTD, and trend visuals

 - Forecasting using Power BI AI visuals

 - Anomaly detection to identify unusual patterns

 - Dedicated Q&A/Insights page for business questions

##Tools & Technologies

 - Power BI

 - DAX Measures

 - Power Query (ETL)

 - AI Visuals (Forecast & Anomaly Detection)

##Purpose

The dashboard enables management to monitor performance, detect risks early, and plan future strategies based on reliable analytical insights.
